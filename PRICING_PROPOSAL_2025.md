# Maritime Onboarding System - Prijsanalyse & Voorstel

## Huidige Situatie

We hebben voor u een volledig op maat gemaakt Maritime Onboarding System ontwikkeld met enterprise-grade functionaliteiten. Het is belangrijk om de waarde van dit systeem in perspectief te plaatsen ten opzichte van de huidige vergoeding.

## Wat is er gebouwd?

### Technische Specificaties
- **Full-stack webapplicatie** met React frontend en Node.js backend
- **Multi-tenant architectuur** voor meerdere bedrijven/schepen
- **Serverless deployment** op Vercel voor optimale schaalbaarheid
- **PostgreSQL database** met Row Level Security via Supabase
- **Real-time email integratie** via MailerSend
- **Automated PDF generatie** voor certificaten met QR-code verificatie
- **Meertalige ondersteuning** (Nederlands/Engels)

### Functionaliteiten
- Volledige 3-fase training workflow (Basic → Advanced → Quiz → Certificaat)
- Interactieve quiz module met randomized vragen
- Role-based access control (Admin/Manager/Crew)
- Magic link authenticatie voor crew members
- Uitgebreide audit logging van alle acties
- Geautomatiseerde herinneringen en notificaties
- Progress tracking en rapportages
- GDPR-compliant data management

### Security & Compliance Features
- JWT authenticatie met refresh tokens
- Rate limiting op alle API endpoints
- Comprehensive security headers (CSP, HSTS, X-Frame-Options)
- Input validatie en sanitization
- Volledige audit trail
- GDPR export en delete functionaliteiten
- Geautomatiseerde backups

## Compliance & Certificering

U heeft aangegeven te willen voldoen aan ISO 27001, NIS2, en GDPR standaarden. De realiteit:

- **ISO 27001 certificering**: €15.000-30.000 (eenmalig) + €5.000-10.000 (jaarlijks)
- **NIS2 compliance audit**: €10.000-20.000
- **Continue compliance monitoring**: €2.000-3.000 per maand

**Voor het huidige budget van €350/maand is formele certificering niet haalbaar.** We kunnen wel werken volgens deze standaarden zonder formele certificering.

## Aanbeveling

Gezien de waarde en complexiteit van het systeem, de vereiste security standaarden, en de noodzaak voor continue onderhoud, adviseren wij minimaal het **Basic Maintenance pakket van €1.500/maand**.

Voor organisaties die serieus zijn over compliance en system reliability, is het **Standard Support pakket van €2.500/maand** meer realistisch.
