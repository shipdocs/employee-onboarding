## Getting Started

### Your First Login

You should have received an email with the subject: **"Begin Your Onboarding Training - Secure Access Link"**

1. **Open the email** and click the **"Start Onboarding"** button (magic link)
2. **You'll be automatically logged in** - no password needed!
3. **Welcome aboard!** You're now in your personal dashboard

**ğŸ” Security Note**: The magic link is secure and expires after 3 hours for your protection.

### If You Can't Find the Email

- Check your **spam/junk folder**
- Search for emails from **noreply@burando.online**
- Contact your manager for a new magic link
- **Note**: Crew members don't use passwords - only secure magic links

- Save as PDF, JPG, or PNG
- Don't use Word docs
- Avoid special characters in filename

**"Upload failed"**
- Check internet connection
- Try different browser
- Clear browser cache
- Contact support if persists

### Taking Assessments

1. **Prepare** before starting:

   - Find quiet location
   - Ensure stable internet
   - Have notepad ready

   - Read questions carefully
   - Don't rush
   - Flag difficult questions

   - See results immediately
   - Review incorrect answers

- **Learn** from incorrect answers

## Troubleshooting Quick Guide

### Login Issues
- Forgot password? Use reset link
- Account locked? Wait 30 minutes
- Wrong email? Check with manager

### Technical Problems
- Clear browser cache
- Try different browser
- Check internet connection
- Disable ad blockers
- Update browser

### Document Issues

- Progress not saving? Don't use private browsing
- Quiz not loading? Refresh page
- Can't hear audio? Check device volume
