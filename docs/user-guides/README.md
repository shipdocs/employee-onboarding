## ğŸ” **Troubleshooting**

### **Common Issues**

#### **Cannot Login**
1. Check email and password spelling
2. Ensure Caps Lock is off
3. Try password reset if needed
4. Contact administrator if account is locked

#### **Magic Link Not Working**
1. Check spam/junk folder
2. Ensure link hasn't expired (valid for 1 hour)
3. Request new magic link
4. Contact support if issue persists

#### **Training Not Loading**
1. Check internet connection
2. Refresh the page
3. Clear browser cache
4. Try different browser
5. Contact support if issue continues

#### **Cannot Upload Files**
1. Check file size (max 10MB for images, 100MB for videos)
2. Verify file format (JPEG, PNG, GIF for images; MP4, WebM for videos)
3. Ensure stable internet connection
4. Try again after a few minutes

### **Performance Issues**
- **Slow Loading**: Check internet speed, try different browser
- **Video Playback**: Ensure sufficient bandwidth, try lower quality
- **Mobile Issues**: Update browser, clear cache, restart device
