üîê Authentication

The API uses JWT (JSON Web Tokens) for authentication. There are two authentication methods:

  • Admin/Manager: Email and password login via /api/auth/login
  • Crew Members: Magic link authentication via /api/auth/request-magic-link

Include the JWT token in the Authorization header for all authenticated requests.

üìç Key Endpoints

  • POST /api/auth/login - Admin/Manager login
  • GET /api/crew/profile - Get crew member profile
  • GET /api/content/training/phases - Get training phases
  • POST /api/training/quiz/{phase}/submit - Submit quiz answers
  • POST /api/upload/content-image - Upload training images
  • GET /api/workflows - Get training workflows
  • GET /api/health - Health check (no auth required)
